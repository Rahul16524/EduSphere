\# EduSphere



EduSphere is an enterprise-level School Management System built to learn real-world software development using Java, Spring Boot Microservices, React, Git, GitHub, and enterprise development practices.



\## Tech Stack



\- Java 17

\- Spring Boot 3.x

\- Spring Cloud

\- React

\- MySQL

\- Maven

\- Git

\- GitHub



\## Microservices



\- Discovery Server

\- Config Server

\- API Gateway

\- Admin Server

\- Auth Service

\- School Service

\- Payment Service
